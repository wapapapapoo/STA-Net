# STA-Net

A novel decoding approach for hybrid EEG-fNIRS BCI.

Work in progress.

The code will be available upon acceptance of the paper.

## Requirements:
- Python 3.9.7
- Tensorflow 2.10

## Dataset

1.[Open access dataset for simultaneous EEG and NIRS brain-computer interface (BCI)](https://doc.ml.tu-berlin.de/hBCI/contactthanks.php)

2.[Simultaneous acquisition of EEG and NIRS during cognitive tasks for an open access dataset](https://doc.ml.tu-berlin.de/simultaneous_EEG_NIRS/)
